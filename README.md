# Swiggy DPDP Demo

Clickable Privacy Center prototype for a Product Management case study on Swiggy's DPDP compliance redesign.

## What This Demonstrates

- A DPDP-ready Privacy Center dashboard for consent, location, marketing, and data rights
- Onboarding consent and granular permission controls
- Marketing preferences, location modes, data download, account deletion, and consent history
- Mock analytics instrumentation with an in-product debug panel
- Documented backend API contract in `API_CONTRACT.md`

## Tech Stack

- Next.js App Router
- React and TypeScript
- Tailwind CSS
- shadcn/ui-style components
- lucide-react icons
- Framer Motion
- Local React state only

## Run Locally

```bash
npm install
npm run dev
```

Then open `http://127.0.0.1:3000`.

## Build

```bash
npm run build
```
